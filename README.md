# puffer（河豚）

#### 介绍
puffer（河豚），致力于企业项目快速开发

项目构建的最终目的，就是根据实际业务场景
挑出可封装重用的业务，降低开发难度，减少代码复写量

参考 Spring/SpringBoot ，支持可以部分功能引入
比如，只使用 db包操作数据库。只使用websocket包实现在线聊天等
