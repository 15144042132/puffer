# puffer（河豚）

#### 介绍
~~~
puffer（河豚），致力于中小型项目快速开发
目的是将日常工作中，复用性价值高，且出现频率高的业务进行封装
并提供丰富的操作函数，降低开发难度，减少代码复写量
~~~
~~~
支持单一Jar包引入
例：
1.只引入db依赖操作数据库
2.只引入websocket包实现在线聊天
3.只引入定时包实现定时任务持久化
4.等等...（完善中）

~~~
