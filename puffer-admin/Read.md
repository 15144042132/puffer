# 模块说明

~~~
common 公共内容
    
module 业务相关
    system 系统模块
        controller 控制器
            DictController 字典管理
            MenuController 菜单管理
            ResourceController 资源管理 
            RoleController 角色管理
            SystemController 用户管理    
        service 服务
            DictController 字典服务
            MenuController 菜单服务
            ResourceController 资源服务
            RoleController 角色服务
            SystemController 用户服务  
        mapper 数据库操作类
~~~
